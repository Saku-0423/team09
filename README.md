# シューティングゲーム

##アプリ概要
プレイヤーがマウスでキャラクターを操作し、敵を倒しながらステージを攻略する横スクロール型シューティングゲームです。
プレイヤーはマウス移動で左右にキャラクターを操作し、マウスクリックで弾を発射します。敵には複数の種類が存在し、それぞれ弱点となる弾が異なるため、敵の特徴に応じた弾薬を使用する必要があります。また、一部には倒すことのできない敵も配置されており、攻撃だけでなく回避しながら進むことも重要となります。
ステージ内では、一時的に無敵になるアイテムや残機を回復するアイテムを取得できます。武器は敵を倒すことでレベルアップし、ピストルからショットガン、ライフルへと進化します。武器の強化に伴い、発射できる弾数や弾速も向上します。
ゲームは全3ステージで構成され、各ステージの最後にはボスが待ち受けています。ボスは戦闘中に弱点が変化するため、状況に応じた攻撃方法が求められます。また、ボスを倒すたびに次のボスの体力が増加し、難易度が徐々に上昇します。
##主な機能
-ウスによる左右移動
-マウスクリックによる弾の発射
-敵ごとに異なる弱点属性
-倒せない敵の配置
-アイテム（無敵・残機回復）
-器レベルアップ（ピストル→ショットガン→ライフル）
-武器強化による弾数・弾速の向上
-全3ステージ
-弱点が変化するボス戦

Magic Shootingの取扱説明書
作　Team9

1.ゲーム概要
Magic Shootingは、属性を切り替えながら敵を倒してステージを攻略するシューティングゲームです。
プレイヤーは魔法使いとなり、火・水・雷の3属性を使い分けながら敵を撃破し、各ステージのボスを倒すことを目指します。
敵やボスにはそれぞれ弱点属性が存在し、弱点を突くことで通常より大きなダメージを与えることができます。
全3ステージをクリアするとゲームクリアです。

2.動作環境
動作環境は以下の通りです
Processing 4
Windows 10 / 11
マウス
キーボード

3.操作方法
操作はマウスとキーボードを使用します。
マウス	プレイヤーを左右に移動
左クリック	魔法弾を発射
スペースキー	魔法属性を切り替える（水→火→雷）

5.プレイヤー
プレイヤーはマウスで左右に移動します。
クリックすると現在装備している武器で魔法弾を発射します。
敵や敵弾に当たるとライフが1減少します。
被弾後は約8秒間無敵になります。
初期ライフは3つです。
ライフが0になるとゲームオーバーになります。

6.属性システム
プレイヤーは3種類の属性（火、水、雷）を自由に切り替えられます。

属性　　有利な敵
水	　　炎の精霊
火	　　氷のゴーレム
雷	　　風の精霊

弱点属性で攻撃すると通常より大きなダメージを与えられます。
ボスにも弱点属性があり、一定時間ごとに変化します。
画面上部の「WEAK POINT」を見ながら属性を切り替えて戦いましょう。

9.ボス
各ステージの最後にはボスが登場します。
ボスは通常の敵よりも高いHPを持っています。

ボスの特徴
自機狙いの属性弾を発射
HPが減ると3方向攻撃を使用
弱点属性が3〜10秒ごとに変化
弱点属性で攻撃すると大ダメージ
画面上部には
ボス名とHPバーと現在の弱点属性が表示されます。

10.アイテム
敵を倒すとアイテムを落とすことがあります。
また、プレイ中にランダムでアイテムが出現することもあります。
❤️ 回復アイテムはライフを1回復します。
ライフは最大3までです。
⭐ 無敵アイテムは取得すると約8秒間無敵になります。
敵や敵弾に当たってもダメージを受けません。

12.攻略のポイント
敵の弱点属性を狙うと効率よく倒せます。
ボス戦では「WEAK POINT」を確認して属性を切り替えましょう。
無敵アイテムはボス戦まで温存できるなら有効です。
ステージ3ではショットガンを活かし、近距離から複数の弾を当てると高いダメージを与えられます。
敵弾はプレイヤーを狙って飛んでくるため、左右への細かい移動で回避すると避けやすくなります。

13.勝利条件
全3ステージをクリアする。

14.ゲームオーバー条件
プレイヤーのライフが0になる。
